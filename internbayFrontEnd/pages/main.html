Posted by TechCorp
Remote
Full Time
$25 per hour
Posted 2 days ago
Job Description

We are seeking a highly skilled Software Developer with expertise in web application development to join our dynamic team. The ideal candidate will have a strong background in ASP.NET, C#, SQL Server, and other relevant technologies.

Responsibilities
  • Develop and maintain web applications using ASP.NET, C#, SQL Server, HTML, CSS, JavaScript
  • Participate in the entire software development lifecycle
  • Collaborate with cross-functional teams to design and implement innovative solutions
  • Utilize data analytics to optimize operational processes
  • Provide exceptional customer service by addressing inquiries and requests
Requirements
  • Bachelor's degree in Computer Science or related field
  • 4+ years of experience in web application development
  • Proficiency in Microsoft Visual Studio and version control tools
  • Strong analytical and problem-solving skills
  • Experience with Agile or Waterfall methodologies
  • Ability to work collaboratively and mentor junior developers
Preferred Qualifications
  • Experience with .NET Core and Bootstrap frameworks
  • Knowledge of remote server management tools
